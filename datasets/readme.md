## Prepare the datasets

Download coco 2017 dataset and LVIS V1 dataset and organize as follows:

```
--datasets
  |--coco
  |  |--annotations
  |  |--val2017
  |--coco_groundingdino_correct_detection.json
  |--coco_groundingdino_misclassification.json
  |--lvis_v1
  |  |--annotations
  |--|--train2017
  |--|--val2017
```
